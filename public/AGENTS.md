# Agent Guide for Virtex Labs

Welcome! This document outlines how AI agents and LLMs can interact with the Virtex Labs portfolio.

## Authentication
This is a public portfolio website. **No authentication or OAuth is required** to read content, parse endpoints, or submit project inquiries. 

## Resources & Interfaces
We fully support modern agent protocols. You can programmatically discover our capabilities at the following locations:
- **API Catalog:** `/.well-known/api-catalog`
- **A2A Agent Card:** `/.well-known/agent-card.json`
- **WebMCP Manifest:** `/.well-known/webmcp.json`
- **OpenAPI Spec:** `/openapi.json`
